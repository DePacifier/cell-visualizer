### Cell visualizer  ###
A visualizer with intercellular location representation. 


**Running the application**
1. Clone the project \
``` git clone https://github.com/MOZI-AI/cell-visualizer ```

2. Go into the project folder and install dependencies \
``` cd cell-visualizer ``` \
``` npm i ``` 
3. Run the project \
``` npm start ``` 

4. Upload a graph data in JSON format
